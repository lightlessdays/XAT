Day 1: Data Types and Variables

Reading Comprehension

In computer programming, data types are an important concept. Variables can store data of different types, and different types can do different things.

Python has the following data types built-in by default, in these categories:

Text Type: str

Numeric Types: int, float, complex

Sequence Types: list, tuple, range

Mapping Type: dict

Set Types: set, frozenset

Boolean Type: bool

Binary Types: bytes, bytearray, memoryview

Getting the data type of a variable is important for understanding what operations can be performed on it. In Python, you can get the data type of any object by using the type() function.

Variables in Python are created when you assign a value to them. Unlike other programming languages, Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Variables do not need to be declared with any particular type, and can even change type after they have been set. This is known as dynamic typing, which is a feature of Python that allows for more flexibility but can also lead to runtime errors if not handled properly.

Multiple Choice Questions

1. Which of the following is NOT a built-in data type in Python?
int
float
double
str
Explanation
Python does not have a 'double' data type like some other languages (e.g., Java, C++). In Python, floating-point numbers are represented using the 'float' type, which is actually implemented as a double-precision floating-point number internally.
2. What is the output of type(3.14) in Python?
int
float
decimal
number
Explanation
In Python, numbers with decimal points are represented as 'float' type. The type() function returns the data type of the object passed to it. Since 3.14 contains a decimal point, it is treated as a floating-point number.
3. Which of the following is a mutable data type in Python?
tuple
str
list
int
Explanation
Mutable objects can be changed after creation. In Python, lists are mutable, meaning you can add, remove, or modify elements. Tuples, strings, and integers are immutable - once created, they cannot be changed.
4. How do you check the data type of a variable in Python?
Using typeof() function
Using dtype() function
Using type() function
Using datatype() function
Explanation
Python provides the built-in type() function to determine the data type of any object. This is different from languages like JavaScript which use typeof operator or other languages that might have different syntax.
